单一变量存放多类型数据

std::variant

std::variant 是 C++17 中引入的一种标准库类型，用于表示一组可能的类型中的一个值。它类似于 C++11 中的 boost::variant，但是更加现代化和易于使用。实际上就是一种类型安全的union，但是内存占用更大，如果不是底层优化之类的应该使用variant，更安全

在使用 std::variant 时，你首先需要包含 <variant> 头文件。std::variant 可以容纳多个不同的类型，并且只占用所包含类型中最大的那个的内存空间。。

可以作为函数返回值，原先的optional作为可选的变量只能返回一个空字符，而如果是variant就可以在读取不成功的时候返回其他信息，例如返回错误代码来帮助修正

std::visit

std::visit是C++17中引入的一个函数模板，它允许你访问存储在std::variant中的值。std::variant是一个可以存储多种不同类型的单一值的类型安全的联合体。使用std::visit，你可以对std::variant`中的当前值执行操作，而不必知道它的确切类型。

std::visit的工作原理是接受一个可调用对象（如函数、lambda表达式或函数对象）和一个或多个std::variant对象。它会调用可调用对象，传递给它当前激活的std::variant中的值。如果有多个std::variant，它会使用它们当前的值来调用可调用对象。
