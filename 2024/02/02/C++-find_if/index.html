find_if

std::find_if 是 C++ 标准库 <algorithm> 头文件中的一个函数，用于在指定范围内查找满足指定条件的第一个元素。它是一个通用的查找算法，非常灵活，可以应用于各种容器和迭代器。

  • first 和 last 是表示查找范围的迭代器，表示一个左闭右开的区间 [first, last)。
  • p 是一个一元谓词（Unary Predicate），是一个可调用的对象，它接受一个参数并返回一个 bool 类型的值。

find_if 函数会在范围 [first, last) 中查找第一个使谓词 p 返回 true 的元素，并返回指向该元素的迭代器。如果没有找到符合条件的元素，返回 last。
