std::function

std::function 是 C++11 引入的一个通用、多态的函数封装器。它可以存储、复制和调用任何可调用对象，包括普通函数、lambda 表达式、函数指针、成员函数指针和函数对象。std::function 提供了一种类型安全的方式来处理这些可调用对象。

优点

  1. 类型安全：std::function 提供了一种类型安全的方式来存储和调用可调用对象。
  2. 多态性：可以存储不同类型的可调用对象（普通函数、lambda 表达式、仿函数、成员函数）。
  3. 灵活性：可以在运行时决定调用哪个函数。

注意事项

  • std::function 对象的参数类型和返回类型必须与被封装的可调用对象匹配。
  • std::function 对象不能用于检查相等性，但可以与 nullptr 进行比较。

std::bind

std::bind 是 C++11 标准库中的一个函数，用于创建一个可调用对象，通常称为“绑定”。这个函数允许你绑定一个或多个函数的参数，以便在稍后的时间点调用它。std::bind 可以与普通函数、成员函数、函数对象等一起使用，非常灵活。

std::mem_fn

std::mem_fn 是 C++11 引入的一个工具，用于生成可以存储、复制和调用成员函数指针的包装器对象。它可以与普通函数、成员函数、函数对象等一起使用，非常灵活。

std::mem_fn 的主要用途是简化成员函数的调用，特别是在需要将成员函数作为参数传递给算法或其他函数时。

std::bind 更灵活，因为它允许你使用占位符来指定参数的位置和数量。这使得 std::bind 在需要部分应用（partial application）或重新排列参数时非常有用。

使用 std::bind 的场景：

  • 需要部分应用：如果你只想绑定部分参数，而将其他参数留给调用时指定。
  • 需要重新排列参数：如果你需要改变参数的顺序。
  • 需要更复杂的绑定逻辑：例如，绑定多个函数或组合多个操作。

使用 std::mem_fn 的场景：

  • 简单调用成员函数：如果你只需要调用成员函数而不需要部分应用或重新排列参数。
  • 代码简洁性：std::mem_fn 更加简洁和直接，适合简单的成员函数调用。
