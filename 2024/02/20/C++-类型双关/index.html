type punning

C++编程中，使用一个类型的数据作为另一个不同类型的数据来处理的情况。这通常涉及到对数据的底层二进制表示的操作和理解。在C++中，你可以通过联合（union）或指针转换等方式实现类型双关。

这些都是c语言风格的类型转换，c++也可以使用reinterpret_cast来进行类型双关解释

eg:Another* aa = reinterpret_cast<Another*>(&value);

举例子，让我们假设有一个浮点数浮点型变量 float f = 12.3; ，我们想看看它在内存中的二进制表示。我们可以通过类型双关来实现这个目标，创建一个整型指针，指向浮点数的内存地址，就可以把浮点数的二进制表示按整型数据来查看了。
